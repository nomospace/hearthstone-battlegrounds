## 📱 技术实现方案

### 数据源
- **英雄数据**: Hearthstone Wiki API
- **实时数据**: HSReplay API
- **卡牌图片**: Blizzard CDN
- **阵容数据**: 社区贡献 + 人工整理

### 技术栈
- **前端**: Angular 17 + Signals
- **状态管理**: NgRx SignalStore
- **图表**: Chart.js / ngx-charts
- **拖拽**: Angular CDK Drag-Drop
- **API**: HttpClient + Interceptors
- **缓存**: IndexedDB (离线可用)

### 性能优化
- **懒加载**: 按需加载模块
- **预加载**: 核心数据预加载
- **缓存策略**: 数据缓存 + 增量更新
- **CDN**: 静态资源 CDN 加速
